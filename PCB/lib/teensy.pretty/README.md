Teensy library for KiCAD
=======================================

This repo includes footprints for the following Teensy versions:

  - Teensy 1.0
  - Teensy++ 1.0
  - Teensy 2.0
  - Teensy++ 2.0
  - Teensy 3.0
  - Teensy 3.1
  - Teensy 3.2
  - Teensy 3.5
  - Teensy 3.6
  - Teensy LC

For symbols look here: https://github.com/XenGi/teensy_library
